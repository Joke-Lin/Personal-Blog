树状数组

树状数组是能够完成下述操作的数据结构：

给定一个初始值全为0的数列，\(a_1,a_2,\dots,a_n\)

  • 给定i，计算\(a_1+a_2+\dots+a_n\)
  • 给定i和x，执行\(a_i+=x\)

即单点修改和区间和计算
