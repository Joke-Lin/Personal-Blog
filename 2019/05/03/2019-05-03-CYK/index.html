类说明

CNF

由一个字典初始化，输入为规范的范式

find_product方法实现输入一个变元组合返回其生成变元

CYK

由一个CNF初始化，find方法用于判断字符串是否在CNF中

并保存推导过程，即动态规划数组

Input

实现自己的输入函数，一直等待输入直到输入EOF（Ctrl + Z）

并对输入字符串进行简单处理，切分

输入字符串格式为：\[A -> BC|AD|b \]

其中空格可含可不含

Visualization

可视化类，将推导过程输出到.csv文件中

将.csv文件使用Excel打开便可查看完整推导过程
